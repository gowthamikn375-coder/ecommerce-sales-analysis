# E-Commerce Sales Analysis

Data Analytics Internship Project 1 — Pluto Academy

## Overview
Analysis of 12 months of e-commerce sales data (Olist Brazilian E-Commerce Dataset from Kaggle) to identify revenue trends, top-performing categories/regions, and customer sentiment patterns.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab

## What's Included
- `ecommerce_sales_analysis.ipynb` — full data cleaning, analysis, and visualizations
- `dashboard.png` — 1-page summary dashboard with key KPIs

## Key Findings
- Top category: Health & Beauty (~R$1.26M revenue)
- Peak sales month: November 2017 (Black Friday season)
- Top region: São Paulo (SP), ~R$5.2M
- 63% of reviews are 5-star; 14.8% are 1-star

See the notebook for full analysis, charts, and business recommendations.
